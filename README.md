# Imagina Pay — Release v1.0.0

**`imagina-pay.zip`** es el plugin final listo para instalar: incluye las
dependencias de producción (Action Scheduler) y el frontend compilado.

## Instalación

1. Descarga `imagina-pay.zip` desde esta rama.
2. En WordPress: **Plugins → Añadir nuevo → Subir plugin** → selecciona el zip → Instalar → Activar.
3. Configura credenciales y webhooks en **wp-admin → Imagina Pay → Ajustes**
   (guía completa y checklist de sandbox en el README de la rama `main`).

Requisitos: PHP 8.1+ · WordPress 6.4+ · permalinks activos.
